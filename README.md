# inventory-management-system-java

Features
- Generic inventory management using Java Generics
- Multiple product categories (Books, Electronics, Clothing)
- Wishlist management
- Recently viewed items tracking
- Price range filtering
- Availability filtering
- Priority-based order processing

Concepts Used
- OOP
- Inheritance
- Polymorphism
- Generics
- Collections Framework
- Comparable
- Comparator
- Priority Queue

Tech Stack
- Java
